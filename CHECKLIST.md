# WBAnalyzer — Чеклист проекта

## СДЕЛАНО

### Инфраструктура
- [x] Python backend, PostgreSQL, MPStats API, Anthropic API
- [x] Мультивалютность (руб/$/€/Br) с реальными курсами ЦБ
- [x] Кэш каталога (6 часов) + кэш MPStats (24 часа)
- [x] Стриминг всех агентов (10 агентов)
- [x] Убраны AI Инсайты (экономия токенов)

### Портфель (полностью реализован)
- [x] Модальное окно перевода из "В работе" с CNY/USD конвертером
- [x] 7 стадий: Лист заказа / Заказано / В пути / Таможня / Склад РБ / На WB / Продаётся
- [x] Крупные вкладки на всю ширину (вариант Б)
- [x] Разные поля для каждой стадии
- [x] Индикатор контейнера Карго/20ft/40ft
- [x] Кнопка "Отправить заявку → перевести в Заказано"
- [x] Баннеры интеграций (МойСклад, 1С, WB API)
- [x] Связь Портфель → Компания (выручка в cashflow)

### Мобильная версия
- [x] Hamburger меню, адаптивная шапка
- [x] Графики в 1 колонку, таблица топ-товаров компактная
